Analyzing which states are waiting on disabality benefits, using R

Social Security Adminstration offices throughout the country approve or disaprove of disability claims called Supplemental Security Income (SSI) or Social Security Disability Insurance United States SSDI. I looked into the number of pending cases for SSI and SSDI in Febuary.

I used a U.S GeoJSON shape file to make this map.

I then merged the shape file with a dataset on pending disablity cases on R Studio.

The skeleton of the map was made using ggplot. I then made it responsive using ai2html.

The map and my analysis showed that Texas had the highest number of pending cases.

NYC Jails are sitting on a landfill that's sinking

My master's thesis looked into how inmates are harmed by the air pollution and heat on Rikers, a 413 acre island housing eight active carceral facilities.

The island suffers from pollution (I have the latest data on this via public records) and because it was built on a landfill, it is sinking at a faster rate than most of NYC.

I made this map using NASA's InSAR data. I then visualized the data on QGIS, cleaned it on Adobe Illustrator and made it responsive using ai2html.
